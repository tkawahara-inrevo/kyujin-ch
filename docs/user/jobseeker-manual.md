# 求職者マニュアル

求人ちゃんねる (https://kyujin-ch.jp) を求職者として利用する方法。

## アカウント作成

1. トップページから「**会員登録**」をクリック
2. メールアドレス・パスワード・氏名を入力
3. 利用規約・プライバシーポリシーに同意 → 登録ボタン
4. メアド認証は不要、即時利用可能

または「**Googleでログイン**」で Google アカウント連携も可能。

## ログイン

- URL: https://kyujin-ch.jp
- 上部右の「**ログイン**」をクリック
- メアド + パスワード or Google ログイン

## 求人を探す

### 一覧から探す

- URL: https://kyujin-ch.jp/jobs
- フィルタ:
  - **キーワード**（職種名・社名等）
  - **都道府県**
  - **求人カテゴリ**（営業・事務・IT等）
  - **雇用形態**（正社員・契約社員・アルバイト等）
  - **年収帯**
- 並び替え: 新着 / 人気 / おすすめ

### 個別求人を見る

- 求人カードをクリック → 詳細ページ
- 表示内容: 仕事内容・給与・勤務地・休日休暇・福利厚生・選考フロー等
- 「**お気に入り**」アイコンで保存

## 応募する

1. 求人詳細ページの下部「**応募する**」ボタン
2. プロフィール情報（自動入力）+ 任意の自己PRを記入
3. 「**応募内容を確認**」→「**応募する**」
4. 応募完了。企業からの返事はメッセージで届く

### 応募状況の確認

- マイページ → **応募管理**
- 各応募のステータス:
  - **未読**: 企業が未確認
  - **既読**: 企業が確認済み
  - **通過**: 選考通過
  - **不採用**: 落選

## メッセージ

- 応募した求人ごとに企業とのメッセージスレッドが自動生成
- マイページ → **メッセージ**
- スレッドを開いて返信送信
- 既読・未読が個別管理

## プロフィール

- マイページ → **プロフィール編集**
- 基本情報・履歴書情報を編集
- アバター画像アップロード可
- 履歴書情報（学歴・職歴・資格・自己PR）を充実させると応募時の入力が楽

## 履歴書ビルダー

- マイページ → **履歴書ビルダー**
- 学歴・職歴・資格を構造化して登録
- PDF ダウンロード可（応募とは別途、自分の控え用）

## お気に入り

- 求人詳細・一覧から ❤️ アイコンクリック
- マイページ → **お気に入り** で一覧確認

## 通報

不適切な求人・企業・メッセージを発見した場合:

- 求人詳細・企業ページ・メッセージから「**通報**」リンク
- URL 直叩き: `/report?type=job&id=<job_id>`
- 通報理由を選択 → 詳細記入 → 送信
- 運営が確認・対応

## ブロック

特定の企業からのメッセージを受け取らないようにする:

- マイページ → **ブロック管理**
- ブロックすると、相手からのメッセージは届かない
- 後から解除可

## 退会

- マイページ → **アカウント削除**
- 確認後、退会完了
- 個人情報は匿名化、応募履歴は企業側で残るが個人特定不可
- メアド・パスワードは即時無効化、再ログイン不可

## トラブル時

- パスワードを忘れた → トップから「**パスワードリセット**」
- ログインできない → お問い合わせフォーム `/contact`
- アカウントロック → 「お問い合わせ」から運営に連絡

## 通知設定

- マイページ → **プロフィール編集** → **通知設定**
- メール通知:
  - 応募ステータス変更時
  - メッセージ受信時
  - 新着求人（任意）
- 通知を全部OFFにすることも可
