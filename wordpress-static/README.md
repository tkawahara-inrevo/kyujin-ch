# WordPress Static Pages

`kyujin-ch.jp` 配下に置く想定の静的HTMLです。

推奨スラッグ:

- `/price`
- `/kiyaku`
- `/kiyaku-company`
- `/application-notes`

対応ファイル:

- `price/index.html`
- `price/prices-data.js`
- `kiyaku/index.html`
- `kiyaku-company/index.html`
- `application-notes/index.html`
- `shared/common.css`

WordPress 側では、各ディレクトリを同名スラッグの固定ページまたは静的公開先に配置して参照してください。
